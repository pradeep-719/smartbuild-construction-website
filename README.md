# SmartBuild Construction Website

SmartBuild is a web-based platform that helps users plan small house construction within their budget. The system suggests suitable contractors and construction materials based on the user’s budget and requirements. The goal of this project is to make construction planning easier, cost-effective, and efficient.

## Features
- Budget based construction planning
- Contractor recommendations
- Construction material suggestions
- User-friendly interface
- Upload and verification of documents

## Technologies Used
- HTML
- CSS
- PHP
- MySQL
- JavaScript

## How to Run the Project
1. Install XAMPP or any local server.
2. Copy the project folder into the `htdocs` directory.
3. Import the database into MySQL.
4. Start Apache and MySQL from XAMPP.
5. Open the project in browser.
